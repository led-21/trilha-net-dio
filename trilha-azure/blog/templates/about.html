Sobre o Blog

Este é um blog de demonstração hospedado na Azure, utilizando os seguintes serviços:

  • Azure App Service ou Container Apps para hospedagem
  • Azure Container Registry (ACR) para armazenamento da imagem Docker
  • Azure Database for PostgreSQL como banco de dados

A aplicação foi desenvolvida em Python com o framework Flask.
